Objectives

In the lab and assignments, you will demonstrate your knowledge of applying the concepts, technology and technique of web site development.

Exams are given to help you consolidate your learning from the class and to provide both student and instructor with an assessment of how you have learned to think about web site development. The exams are designed to test higher order learning, i.e., your ability to apply what youâ€™ve learned, not simple memorization of facts.

Learning Outcomes

By the end of this course students should be able to:
  1. Apply web development tools such as HTML5 and CSS effectively to build web pages.
  2. Use text, color, graphics, and media effectively on a web page.
  3. Understand how to effectively organize a web site.
  4. Understand how to effectively present information on the web.
  5. Understand ethical and legal considerations encountered when working in web development.
